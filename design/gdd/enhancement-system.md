# Enhancement System (강화 시스템)

*Created: 2026-03-27*
*Status: Draft*
*Dependencies: [hero-collection](hero-collection.md), [wave-defense](wave-defense.md), [economy](economy.md)*

---

## 1. Overview

리니지클래식의 강화 성공/실패 도박을 풍운지기의 영웅 성장 메타게임에 접목한 시스템.
영웅을 +1~+10까지 강화할 수 있으며, 단계가 올라갈수록 성공률이 낮아지고
실패 시 단계가 하락한다. 강화 시도 자체가 "한 번만 더"의 중독적 루프를 만들며,
TD 웨이브 파밍의 강력한 동기를 제공한다.

강화는 **심화 옵션(depth layer)** 이지, 콘텐츠 게이팅 도구가 아니다.
일반 스테이지는 +0 영웅으로 클리어 가능해야 한다.

## 2. Player Fantasy

> "내가 직접 키운 +7 도사가 금빛 오라를 두르고 전장에 서 있다.
> 모루 위에서 망치를 내려치는 그 순간, 심장이 멈추는 것 같았는데...
> 성공했다! 화면 전체가 빛나며 축하 메시지가 뜬다.
> 이 도사는 이제 내 최고의 영웅이다."

플레이어는 **도박사이자 장인**이다. 재료를 모아 강화에 도전하고,
성공의 짜릿함과 실패의 아쉬움을 반복하며, 결국 자신만의 전설적 영웅을 만든다.

## 3. Detailed Rules

### 3.1 강화 대상

- 영웅 자체를 직접 강화 (장비 시스템 없음)
- 각 영웅은 독립적인 강화 단계를 보유 (+0 ~ +10)
- 강화 상태는 영웅에 영구 귀속

### 3.2 강화 구간

| 구간 | 범위 | 특성 |
|------|------|------|
| **안전 구간** | +1 ~ +4 | 실패해도 단계 유지, 재화만 소모 |
| **위험 구간** | +5 ~ +7 | 실패 시 -1 단계 하락 |
| **지옥 구간** | +8 ~ +10 | 실패 시 -2 단계 하락 |

### 3.3 강화 확률표

| 시도 | 성공률 | 실패 결과 | 구간 |
|------|--------|-----------|------|
| +0 → +1 | 100% | — | 안전 |
| +1 → +2 | 100% | — | 안전 |
| +2 → +3 | 90% | 재화만 소모 | 안전 |
| +3 → +4 | 80% | 재화만 소모 | 안전 |
| +4 → +5 | 70% | 재화만 소모 | 안전 |
| +5 → +6 | 55% | -1 단계 (→+4) | 위험 |
| +6 → +7 | 40% | -1 단계 (→+5) | 위험 |
| +7 → +8 | 25% | -2 단계 (→+5) | 지옥 |
| +8 → +9 | 15% | -2 단계 (→+6) | 지옥 |
| +9 → +10 | 8% | -2 단계 (→+7) | 지옥 |

### 3.4 하한선 규칙

- 강화 단계는 **절대 +4 아래로 떨어지지 않음** (+5 이상 달성 경험이 있는 경우)
- 최초 강화 시작 시점의 바닥은 +0

### 3.5 불운 축적 (Pity System)

- +6→+7 시도: **5회 연속 실패** 후 다음 시도 성공률 +20% 보너스
- +9→+10 시도: **10회 연속 실패** 후 성공 보장
- 불운 게이지는 UI에 공개 표시 (투명성)
- 해당 단계 성공 시 게이지 리셋

### 3.6 보호 아이템

| 아이템 | 효과 | 적용 구간 | 획득처 |
|--------|------|-----------|--------|
| **강화 부적** | 실패 시 단계 하락 방지 (재화만 소모) | +5 ~ +7 | 스테이지 3성 클리어, 챕터 완료 |
| **천상의 축복** | 실패 시 -1만 하락 (기본 -2 대신) | +8 ~ +10 | 하드 모드 보스, 도전 스테이지 |
| **강화 성공 주문서** | 다음 시도 성공률 +15% | 전 구간 | 영웅 도감 달성, 이벤트 |

- 모든 보호 아이템은 소모성 (1회 사용 후 소멸)
- 보호 아이템은 **게임플레이로만 획득** (과금 루트 없음)

### 3.7 실패 보상

- 강화 실패 시 **"영혼의 파편"** 소량 지급
- 파편 10개 → 강화 부적 1개 교환 가능
- 실패가 완전한 낭비가 아니라는 심리적 완충 장치

## 4. Formulas

### 4.1 강화 비용 공식

```
강화_골드_비용 = base_gold × cost_multiplier[level]
강화석_비용 = base_stones × stone_multiplier[level]
```

| 변수 | 값 | 설명 |
|------|-----|------|
| `base_gold` | 100 | 기본 골드 비용 |
| `base_stones` | 1 | 기본 강화석 비용 |

| 단계 | cost_multiplier | stone_multiplier | 골드 비용 | 강화석 (등급) |
|------|----------------|-----------------|-----------|-------------|
| +1 | 0.5 | 1 | 50 | 하급 1개 |
| +2 | 1.0 | 1 | 100 | 하급 1개 |
| +3 | 2.0 | 2 | 200 | 하급 2개 |
| +4 | 4.0 | 3 | 400 | 하급 3개 |
| +5 | 7.0 | 5 | 700 | 하급 5개 |
| +6 | 12.0 | 2 | 1,200 | 중급 2개 |
| +7 | 20.0 | 3 | 2,000 | 중급 3개 |
| +8 | 35.0 | 5 | 3,500 | 중급 5개 |
| +9 | 60.0 | 2 | 6,000 | 상급 2개 |
| +10 | 100.0 | 3 | 10,000 | 상급 3개 |

### 4.2 강화 스탯 보너스 공식

```
최종_스탯 = 기본_스탯 × atk_multiplier[level]
최종_사거리 = 기본_사거리 × range_multiplier[level]
최종_쿨다운 = 기본_쿨다운 × (1 - cooldown_reduction[level])
```

| 단계 | atk_multiplier | range_multiplier | cooldown_reduction |
|------|---------------|-----------------|-------------------|
| +0 | 1.00 | 1.00 | 0.00 |
| +1 | 1.05 | 1.00 | 0.00 |
| +2 | 1.10 | 1.00 | 0.00 |
| +3 | 1.16 | 1.03 | 0.00 |
| +4 | 1.22 | 1.05 | 0.00 |
| +5 | 1.30 | 1.08 | 0.05 |
| +6 | 1.40 | 1.10 | 0.08 |
| +7 | 1.55 | 1.13 | 0.12 |
| +8 | 1.75 | 1.16 | 0.16 |
| +9 | 2.00 | 1.20 | 0.20 |
| +10 | 2.40 | 1.25 | 0.25 |

**예시 계산 — 궁수(기본 ATK 30, 사거리 200, 쿨다운 8초) +7 강화 시:**
- 공격력: 30 × 1.55 = 46.5 → 47 (반올림)
- 사거리: 200 × 1.13 = 226
- 쿨다운: 8 × (1 - 0.12) = 7.04초

### 4.3 기대 비용 계산 (Expected Cost)

```
기대_시도_횟수[level] = 1 / 성공률[level]
기대_총_비용 = Σ (기대_시도_횟수[i] × 비용[i])  for i = current to target
```

**예시 — +0에서 +7까지 기대 비용:**

| 단계 | 기대 시도 | 골드/시도 | 기대 골드 |
|------|----------|----------|----------|
| +1 | 1.0 | 50 | 50 |
| +2 | 1.0 | 100 | 100 |
| +3 | 1.1 | 200 | 222 |
| +4 | 1.25 | 400 | 500 |
| +5 | 1.43 | 700 | 1,000 |
| +6 | 1.82 | 1,200 | 2,182 |
| +7 | 2.50 | 2,000 | 5,000 |
| **합계** | | | **~9,054** |

(하락 후 재강화 비용 미포함 — 실제 기대 비용은 +6~+7 구간에서 약 1.5~2배 증가)

## 5. Edge Cases

| 상황 | 처리 |
|------|------|
| 골드는 있지만 강화석이 부족할 때 | 강화 버튼 비활성화, "강화석이 부족합니다" 메시지 + 획득처 안내 |
| 보호 아이템 사용 중 성공한 경우 | 보호 아이템 소모되지 않음 (성공 시 보호 불필요) |
| +10 영웅에 강화 시도 | 강화 버튼 비활성화, "+10 최대 강화 달성" 표시 |
| 강화 중 게임 종료/크래시 | 강화 결과는 시도 즉시 확정 후 저장. 연출만 재생 중이었다면 결과 반영됨 |
| 불운 게이지 쌓인 상태에서 다른 단계 강화 | 불운 게이지는 단계별 독립. +6→+7 게이지와 +7→+8 게이지는 별도 |
| 동일 영웅 중복 획득 시 | 중복 영웅은 강화석으로 변환 (수집 시스템과 연동) |
| 강화 부적 + 천상의 축복 동시 사용 | 동시 사용 불가, 1회 1개만 장착. 더 높은 등급 우선 표시 |
| +5 이상에서 하한선 +4에 도달 | 이후 실패해도 +4 유지 (더 이상 하락 없음) |

## 6. Dependencies

### 이 시스템이 의존하는 시스템

| 시스템 | 의존 내용 |
|--------|-----------|
| **영웅 수집/성장** | 강화 대상 영웅 데이터, 기본 스탯 |
| **경제 시스템** | 골드 파우셋/싱크 밸런스, 강화석 드롭률 |
| **웨이브 방어** | 강화석 드롭 소스, 골드 획득 소스 |
| **스테이지 진행** | 3성 보상 (강화 부적), 하드모드 보상 (천상의 축복) |
| **세이브 시스템** | 강화 단계/불운 게이지/보호 아이템 영속 저장 |

### 이 시스템에 의존하는 시스템

| 시스템 | 의존 내용 |
|--------|-----------|
| **전투 밸런스** | 강화된 영웅의 스탯이 전투 난이도에 영향 |
| **UI 시스템** | 강화 UI, 불운 게이지, 보호 아이템 슬롯, 연출 |
| **VFX 시스템** | 강화 단계별 오라/파티클, 강화 성공/실패 연출 |

## 7. Tuning Knobs

| 파라미터 | 현재 값 | 안전 범위 | 영향 |
|---------|--------|----------|------|
| `success_rate[level]` | 위 확률표 참조 | 5%~100% | 낮추면 강화 난이도 상승, 골드 싱크 증가 |
| `cost_multiplier[level]` | 위 비용표 참조 | 0.1~200.0 | 높이면 강화 접근성 감소, 파밍 시간 증가 |
| `atk_multiplier[level]` | 위 스탯표 참조 | 1.0~3.0 | 높이면 강화 보상감 증가, 밸런스 위험 |
| `pity_threshold_67` | 5회 | 3~10회 | 낮추면 +7 접근성 증가 |
| `pity_threshold_910` | 10회 | 5~20회 | 낮추면 +10 접근성 증가 |
| `pity_bonus` | +20% | +10%~+50% | 높이면 불운 보정 강해짐 |
| `floor_level` | +4 | +0~+6 | 높이면 하락 좌절감 감소, 도박 긴장감도 감소 |
| `fragment_per_fail` | 1개 | 0~3개 | 높이면 실패 보상감 증가, 부적 접근성 증가 |
| `fragments_per_talisman` | 10개 | 5~20개 | 낮추면 보호 아이템 접근성 증가 |

## 8. Acceptance Criteria

### 핵심 기능

- [ ] 영웅 선택 후 강화 UI에서 현재 단계, 성공률, 비용을 확인할 수 있다
- [ ] 강화 시도 시 확률표에 따라 성공/실패가 결정된다
- [ ] 안전 구간(+1~+4) 실패 시 단계가 유지되고 재화만 소모된다
- [ ] 위험 구간(+5~+7) 실패 시 -1 단계 하락한다
- [ ] 지옥 구간(+8~+10) 실패 시 -2 단계 하락한다
- [ ] 단계가 +4 아래로 절대 떨어지지 않는다 (+5 이상 경험 시)
- [ ] +10 도달 시 강화 버튼이 비활성화된다

### 보호 시스템

- [ ] 강화 부적 사용 시 +5~+7 실패해도 단계 유지된다
- [ ] 천상의 축복 사용 시 +8~+10 실패 시 -1만 하락한다
- [ ] 강화 성공 주문서 사용 시 성공률이 +15% 증가한다
- [ ] 보호 아이템은 성공 시 소모되지 않는다
- [ ] 보호 아이템 동시 사용이 차단된다

### 불운 축적 (Pity)

- [ ] +6→+7 연속 5회 실패 후 +20% 보너스가 적용된다
- [ ] +9→+10 연속 10회 실패 후 100% 성공이 보장된다
- [ ] 불운 게이지가 UI에 실시간 표시된다
- [ ] 성공 시 해당 단계 게이지가 리셋된다

### 연출

- [ ] 강화 시도 시 모루 + 망치 타격 애니메이션이 재생된다
- [ ] 성공 시 빛 폭발 + 금색 텍스트 + 카메라 흔들림이 재생된다
- [ ] 실패 시 어두운 연기 + 붉은 텍스트가 재생된다
- [ ] +7 달성 시 황금 배너 알림이 3초간 표시된다
- [ ] +10 달성 시 전체 화면 금빛 이펙트 + 전설 알림이 표시된다

### 시각 효과

- [ ] +1~+3: 흰빛 테두리가 영웅에 표시된다
- [ ] +4~+6: 청색 오라가 영웅에 표시된다
- [ ] +7: 금빛 오라 + 파티클 이펙트가 표시된다
- [ ] +8~+9: 붉은 화염 오라 + 지속 파티클이 표시된다
- [ ] +10: 무지개 오라 + 빛 방사 + 별 아이콘이 표시된다

### 경제 연동

- [ ] 강화 시 골드와 강화석이 정확한 비용만큼 차감된다
- [ ] 재화 부족 시 강화 버튼이 비활성화되고 안내 메시지가 뜬다
- [ ] 실패 시 영혼의 파편이 지급된다
- [ ] 파편 10개로 강화 부적 교환이 가능하다

### 밸런스

- [ ] +0 영웅 4종으로 일반 스테이지 1~3 클리어가 가능하다
- [ ] +10 영웅의 공격력이 기본의 2.4배를 초과하지 않는다
- [ ] 강화 비용의 골드 싱크가 웨이브 파우셋의 40~60% 범위 안에 있다

### 강화 연출 시퀀스

```
[시도] → 화면 어두워짐 (0.3s)
      → 모루 등장 + 영웅 올라섬 (0.5s)
      → 망치 타격 1 + 불꽃 (0.4s)
      → 망치 타격 2 + 불꽃 (0.4s)
      → 망치 타격 3 + 불꽃 (0.4s)
      → 정지 (0.5s) ← 긴장 최대
      → [성공] 빛 폭발 + 사운드 + 텍스트 (1.5s)
      → [실패] 연기 + 둔탁한 소리 + 텍스트 (1.0s)
총 소요: 약 3.5 ~ 4.0초
```
